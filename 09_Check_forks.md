# Compare All Forks Script

## Описание

Этот PowerShell-скрипт автоматически получает список всех форков указанного репозитория на GitHub, сравнивает их с основной веткой (master/main) оригинального репозитория и формирует текстовый отчёт с коммитами, которых нет в основной ветке.

Скрипт также отображает прогресс обработки форков через индикатор выполнения.

---

## Конфигурация

В начале скрипта задаются основные параметры:

```powershell
$RepoOwner = "LibreOffice"   # Владелец репозитория
$RepoName  = "core"           # Название репозитория
$PerPage   = 100              # Количество форков на одну страницу API
$UserAgent = "PowerShellScript" # User-Agent для API запросов
